# Resolution-Hardware-Week-Six

For this week I made a distance sensing device in Wokwi. Using an ultrasonic sensor, Raspberyy pi pico and an OLED screen I'm able to display a distance on the screen. 

I chose to do this specific project as I have actually done it before. For a school assignment I made a cornhole game that tracks if the beanbag lands in the hole using an ultrasonic sensor. I was extremely dissapointed with that project as it was just generally bad. Sometimes it just wouldn't display and the wiring was a mess. So I though I eould try again and make it better. This time instead of giving a binary output of either the beanbag is in the hole or not, it provides a rough distance. The distance is about 2cm off each time but I have no clue why or how to fix it. 

This project went quite smoothly as I have enough micropython experience to understand what I'm doing.
